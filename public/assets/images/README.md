# Website images

Drop the school photos here using these exact file names. Until a file is
present, the site shows a branded placeholder automatically — nothing breaks.

## Home & about
- `students-group.jpg` — hero photo (the 5 students). Portrait/tall works best.
- `director.jpg` — the director / chairman's portrait.
- `principal.jpg` — the principal's portrait.

## Facilities (landscape ~16:11)
- `facility-lab.jpg`, `facility-ict.jpg`, `facility-library.jpg`,
  `facility-sports.jpg`, `facility-arts.jpg`, `facility-clinic.jpg`

## Gallery (any size; masonry adapts)
- `gallery-1.jpg` … `gallery-6.jpg`

## Faculty (square headshots)
- `faculty-1.svg` … `faculty-6.svg` (replace with real `faculty-1.jpg`, etc.,
  and update the paths in the admin dashboard or `seed-content.ts`).

## Tips
- Keep files under ~400 KB each (compress at squoosh.app) for fast loading.
- Admins can also upload images from the dashboard once Firebase Storage is on;
  those are stored in the cloud and don't need to live in this folder.
