# Downloadable documents

Place PDF files here to make them available on the **Downloads** page
(`/resources`). Use these names to match the built-in list:

- `brochure.pdf` — school brochure 2026–2027
- `academic-calendar.pdf` — academic calendar 2026–2027

You can also upload/replace these from the **admin dashboard → Downloads**
(stored in Cloud Storage once Firebase is connected — no need to touch this
folder in that case).

Any resource whose file is missing shows "Coming soon" on the page instead of a
broken link.
